# Cause Esclusione DLGS 36_2023
Pagina web interattiva per consultare le cause di esclusione dagli appalti pubblici (artt. 94, 95, 98 D.Lgs. 36/2023) e i reati presupposto del D.Lgs. 231/2001, organizzati in schede navigabili. Include ricerca globale con evidenziazione risultati e tooltip esplicativi sui riferimenti normativi ANAC.
